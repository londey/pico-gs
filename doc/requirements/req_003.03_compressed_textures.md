# REQ-003.03: Compressed Textures

Status: Deleted

This requirement is wholly superseded by the INDEXED8_2X2 single-format architecture.
BC1/BC2/BC3/BC4/BC5 block-compressed formats are removed; the INDEXED8_2X2 palette-indexed format (REQ-003.06) replaces all compressed texture storage.
UNIT-011.04 (Block Decompressor) is deleted.
