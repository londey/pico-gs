# REQ-014.04: Particle System

## Classification

- **Priority:** Important
- **Stability:** Draft
- **Verification:** Test

## Requirement

When the render executor processes a particle system render task, the system SHALL generate and submit camera-facing billboard quads (two triangles each) for each active particle, with per-particle position, size, color, and texture coordinates.

## Rationale

Particle systems enable visual effects (sparks, smoke, explosions, magic effects) by rendering large numbers of small, short-lived, camera-facing textured quads.
The host CPU manages particle lifetimes and physics; the GPU renders the resulting billboards as standard textured triangles.
