# REQ-003.04: Swizzle Patterns

Status: Deleted

This requirement is wholly superseded by the INDEXED8_2X2 single-format architecture.
With INDEXED8_2X2, each palette entry directly provides 4 UQ1.8 RGBA color values; there is no decoded multi-channel texture output to swizzle.
Channel reordering for palette entries is not specified in the INDEXED8_2X2 architecture.
